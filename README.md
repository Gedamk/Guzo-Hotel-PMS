# Guzo Booking (Private)

Private hospitality automation project:
- Booking sync
- Notifications (Email/SMS/WhatsApp/Telegram)
- Payments (Stripe/Telebirr/PayPal - modular)
- Health checks, retries, and webhooks

## Setup
1) Create `.env` from `.env.example`
2) Install deps: `pip install -r requirements.txt`
3) Run: `python run_booking_bot.py`
